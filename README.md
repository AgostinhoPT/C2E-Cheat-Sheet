# Cálculo II E (C2E) - Folha de Resumos 2025/2026

Este repositório contém uma folha de resumos (*cheat sheet*) compacta e completa para a unidade curricular de **Cálculo II E** na FCT-UNL.

## Propósito
O objetivo principal é compilar detalhadamente toda a matéria lecionada nos *slides* teóricos em apontamentos condensados de consulta rápida, garantindo que nenhum tópico ou fórmula seja omitido. Este repositório é **público** com o intuito de ajudar futuros estudantes da disciplina na preparação para os exames e testes. Façam o que bem quiserem com este material, alterem como quiserem e partilhem com quem acharem que possa beneficiar. \\

Quando passar a cadeira (deus queira que seja agora 06/2026) este repo fica morto então dêem fork se quiserem manter o material atualizado.
